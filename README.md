# Shard by Dynexus

Modern Discord Bot Hosting Platform powered by Dynexus.

## ✨ Features

- Free & Premium Discord Bot Hosting
- Credit-Based Renewals
- Grace Period System
- Fast Deployment
- Modern Dashboard
- Pterodactyl Integration
- Dedicated IP & VPS Options
- Abuse Protection & Moderation
- Multiple Hosting Locations

---

## 🛠️ Tech Stack

- Next.js
- TypeScript
- MongoDB
- TailwindCSS
- Pterodactyl
- Docker

---

## 🚀 Setup

```bash
git clone https://github.com/mediax1/Shard.git
cd shard
npm install
npm run dev
```
